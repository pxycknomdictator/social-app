# full Stack social media application

## Features

- User Authentication: Secure login and registration with JWT.
- Post Creation: Users can create posts.
- Comments: Engage with posts through comments.
- Responsive Design: Fully optimized for mobile and desktop views.

## Tech Stack (MERN)

- Frontend: React.js
- Backend: Node.js, Express.js
- Database: MongoDB

## Requirements

- Node.js version: 22.x
- npm version: 10+ (or use pnpm)

## Installation and Setup

## Clone the repository

git clone https://github.com/pxycknomdictator/social-app.git
cd social-app

## Install dependencies

```bash
cd backend
npm install

cd frontend
npm install
```

## Run the development servers

Both frontend and backend commands are same

```bash
npm run dev
```

## If using Docker or Docker-compose

Ensure environment variables are set in constants.js for both backend and frontend

```bash
docker-compose up -d
```

# Development servers
